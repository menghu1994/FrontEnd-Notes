# `!!` Convert to boolean

1. !![]  		// true    
	![]  		// false
2. !!{}     	// true                                                  	!{}          // false
3. !!false      // false 
	!false     	// true 
4. !!true   	// true 
	!true   	// false
5. !!undefined  // false
	!undefined  // true 
6. !!null   	// false  
	!null       // true 
7. !!""   		// false                                         
	!""         // true
